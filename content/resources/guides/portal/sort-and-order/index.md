> This user guide uses 'portal' to refer to the interface across all of the available <GlossaryLink display="Oni" id="oni" /> portals (see [Available Portals](/working-with-data/find-access/#available-portals) for more details).

<br>

The portal search engine allows you to sort and order all items of a query according to the below options. By default, items are sorted by Collections first, and are in Descending order.

## Sort By

- Collections
- Relevance
- Name

<br>

## Order By

- Ascending
- Descending
